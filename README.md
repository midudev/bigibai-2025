## Desarrollo local

1. Clona el repositorio:

    ```bash
    git clone https://github.com/midudev/bigibai-2025.git
    ```

2. Instala las dependencias:

    ```bash
    pnpm install
    ```

3. Inicia el servidor de desarrollo:

    ```bash
    pnpm dev
    ```

4. Abre [http://localhost:4321](http://localhost:4321) en tu navegador.
